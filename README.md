# vue2 项目 vite 构建打包

## 技术栈

- Vue2.7.0

## 构建工具

- vite

## node 版本

- Node.js 18 / 20+

## 启动

```bash
# 进入项目目录
cd vue2VSvite

# 安装依赖
npm install
#建议使用yarn安装依赖
# yarn install

# 运行开发环境
npm run dev

# 打包生产
npm run build

# 打包测试
npm run build:release
```
